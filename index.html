C++ Ray Tracer
C++ · Jan 2026 – Mar 2026 · Graphics Engineering

The engine renders complex scenes with full Phong illumination, recursive mirror reflections using the Schlick–Fresnel approximation, shadow testing, and polygon mesh parsing from Polyray scene files. C++ UNO Engine
C++20 · 2024–2026 · Game Dev · Systems

A fully playable 2–4 player UNO game built in C++20 with genuine software design depth. Features a polymorphic card hierarchy with an abstract base class and virtual methods, a custom circular doubly linked list for turn management, and a hand-coded ANSI RGB terminal UI with per-card color theming — because the visual experience of a program matters just as much as what's running underneath it. Over 1,400 lines of original code.
